# Dyanmic-Bookstore-Website
Project Overview

The Dynamic Web Store is a full-featured e-commerce web application that allows users to browse products, view details, manage a shopping cart, place orders, and manage user accounts. The system is database-driven and dynamically generates content based on user interactions and stored data.
This project demonstrates the use of modern web development technologies, server-side programming, and relational database integration to build a functional online shopping platform.

# Features
User registration and login
Product listing with categories and search
Product details page with images and descriptions
Shopping cart (add, update, remove items)
Order placement and order history
Responsive and user-friendly interface
Secure data handling and validation
